# Geospatial Mapping Operations & Quality Analysis

## Project Overview

This project analyzes geospatial mapping operations data to evaluate mapping productivity, feature distribution, and quality control performance.

The project demonstrates an end-to-end data analytics workflow including data cleaning, SQL analysis, and dashboard visualization.

## Tools Used

* Excel – Data cleaning and preprocessing
* SQL – Data analysis and business queries
* Power BI – Interactive dashboard and visualization

## Dataset

The dataset simulates mapping operations including feature mapping tasks, mapper activity, and quality control reviews.

Main tables:

* Map Features
* Mapping Tasks
* Quality Checks

## Key Analysis

SQL queries were used to analyze:

* Feature distribution by type
* City-wise mapping activity
* Mapper productivity
* QC approval vs rejection rates
* QC engineer workload
* Mapping trends over time

## Dashboard Insights

The Power BI dashboard provides insights into:

* Total mapped features
* Total mapping tasks
* QC approval rate
* Mapper efficiency
* City-wise mapping performance

## Project Workflow

Raw Mapping Data
→ Excel Data Cleaning
→ SQL Data Analysis
→ Power BI Dashboard Visualization
